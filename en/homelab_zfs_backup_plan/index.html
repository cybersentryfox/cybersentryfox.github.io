üì¶ ZFS Storage Setup

Pool Creation

zpool create -o ashift=12 tank /dev/sdb
zfs create tank/vmstore
zfs create tank/iso
zfs create tank/backup
zfs set compression=lz4 tank

Proxmox Storage Configuration

  • zfs-tank (vmstore): VM disk images
  • zfs-tank-iso: ISO storage
  • zfs-tank-backup: Backup location

üîÅ Auto Snapshots

Install Auto Snapshot

Retention Policy

üíæ Backup Plan

üîå Local USB Drive

  • Formatted as ext4 or ZFS
  • Mounted at /mnt/usb-backup
  • Proxmox backup storage added with:
    • ID: usb-backup
    • Content: backup

üîÅ Backup Schedule

  • Weekly full backup to USB
  • Proxmox scheduled backups
  • Manual snapshot exports (optional)

üîê Encrypted ZFS (Optional)

zpool create -O encryption=on -O keylocation=prompt -O keyformat=passphrase securepool /dev/sdc
  • AES-256-GCM encryption at rest
  • Ideal for protecting sensitive data
  • Must not lose passphrase

üìå Notes
