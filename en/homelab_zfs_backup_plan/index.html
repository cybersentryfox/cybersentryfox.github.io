🔐 Cybersecurity Homelab (2025) - ZFS, pfSense, ELK Backup Plan

🖥️ Hardware

  • Dell OptiPlex 5060 SFF
  • Intel i7-8700, 32GB RAM
  • 1TB NVMe (Proxmox OS + ZFS)
  • External USB Drive for Backups

📦 ZFS Storage Setup

Pool Creation

zpool create -o ashift=12 tank /dev/sdb
zfs create tank/vmstore
zfs create tank/iso
zfs create tank/backup
zfs set compression=lz4 tank

Proxmox Storage Configuration

  • zfs-tank (vmstore): VM disk images
  • zfs-tank-iso: ISO storage
  • zfs-tank-backup: Backup location

🧭 Network and VMs

   [ INTERNET ]
        |
     [ pfSense VM ]
        |
   ┌────┼────┬────┐
   |         |    |
[Kali]   [DVWA]  [ELK Stack]
  |         |        |
Snapshot   Snapshot  Daily backup
  • pfSense: Firewall and router
  • Kali Linux: Pentesting
  • DVWA: Vulnerable web app
  • ELK Stack (Ubuntu): Log analysis & monitoring

🔁 Auto Snapshots

Install Auto Snapshot

apt install zfs-auto-snapshot -y

Retention Policy

  • Hourly (24 retained)
  • Daily (7 retained)
  • Weekly (4 retained)
  • Monthly (12 retained)

💾 Backup Plan

🔌 Local USB Drive

  • Formatted as ext4 or ZFS
  • Mounted at /mnt/usb-backup
  • Proxmox backup storage added with:
    • ID: usb-backup
    • Content: backup

🔁 Backup Schedule

  • Weekly full backup to USB
  • Proxmox scheduled backups
  • Manual snapshot exports (optional)

🔐 Encrypted ZFS (Optional)

zpool create -O encryption=on -O keylocation=prompt -O keyformat=passphrase securepool /dev/sdc
  • AES-256-GCM encryption at rest
  • Ideal for protecting sensitive data
  • Must not lose passphrase

📌 Notes

  • Use snapshot rollback before exploits
  • Keep USB offline when not backing up
  • Consider expanding to PBS/NFS in future
